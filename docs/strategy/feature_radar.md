# Feature Radar: Strategy & Roadmap

## 1. Core (MVP) - "The Ignition Phase"
*   **Authentication:** Supabase Auth (Google/GitHub).
*   **Input:** Voice Recording & Transcription (Deepgram).
*   **Logic:** Basic Velocity Calculation (Analyst Agent).
*   **Output:** Home Dashboard with "Velocity Score".

## 2. Horizon 1 - "The Acceleration Phase"
*   **Search:** RAG pipeline (Vector Search) to find past work.
*   **Team Mode:** Manager View (Aggregated Velocity).
*   **Integrations:** GitHub (Auto-import PRs).

## 3. Horizon 2 - "The Autonomy Phase"
*   **Proactive Coaching:** Analyst nudges user: "You haven't done deep work in 3 days."
*   **Career Simulator:** "What if I get promoted?" scenario planning.
